# SmartConvert

SmartConvert е лек, портативен конвертор на файлове за Windows. Добавя опция "Convert" директно в контекстното меню (десен клик), през която можеш да конвертираш изображения, аудио файлове и текстови/код файлове без да отваряш отделна програма.

## Какво прави

- Добавя бутон **Convert** при десен клик на поддържан файл
- Показва само валидните опции за конвертиране, спрямо типа на файла
- Работи изцяло локално, без интернет връзка след първоначалната инсталация
- Не съдържа реклами и не изисква регистрация

## Поддържани формати

**Изображения**
HEIC, PNG, JPG, JPEG, WebP, BMP, TIFF
(HEIC файловете могат да се конвертират само навън, тъй като форматът е затворен)

**Аудио**
MP3, WAV, FLAC, AAC, OGG, M4A

**Текст и код**
TXT, PY, BAT, JSON, MD

## Инсталация

1. Увери се, че имаш инсталиран Python (с отметка "Add Python to PATH" по време на инсталацията)
2. Разархивирай папката
3. Стартирай файла `INSTALLER.py`
4. Изчакай процеса да приключи. Той автоматично:
   - Инсталира нужните библиотеки
   - Билдва програмните файлове
   - Сваля и подготвя ffmpeg за аудио конверсии
   - Подрежда всичко в чиста структура от папки
5. След приключване, можеш да изберем да отвориш SmartConvert директно

## Употреба

1. Десен клик на поддържан файл
2. Избери **Convert**
3. Избери целевия формат от списъка
4. Готово, новият файл се появява в същата папка

## Контролен панел

Стартирай `smartconvert.exe` по всяко време за да:
- Видиш статус на инсталацията
- Включиш или изключиш контекстното меню чрез превключвателя
- Достъпиш деинсталацията

## Деинсталация

Стартирай `uninstall.exe` (или бутона "Деинсталирай" в приложението). Това премахва контекстното меню от Windows и изтрива цялата програмна папка автоматично.

## Технически бележки

- Програмата не записва никакви данни извън собствената си папка
- Аудио конверсиите се извършват чрез вградения ffmpeg
- Изображенията се обработват чрез библиотеката Pillow
- Цялата инсталация е самостоятелна (portable) и не пипа системни настройки извън контекстното меню на Windows.
