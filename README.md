# SmartConvert v1.0

## Конвертиране 
Десен клик на файл → Convert → избери формат (само валидните опции се показват)

## Инсталация
1. Python от python.org (✔ Add Python to PATH)
2. пусни python INSTALLER.py, изчакай да билдне (сваля ffmpeg + всичко необходимо автоматично)

## Деинсталация
Стартирай `uninstall.exe` → изтрий папката ръчно.
